Terminal-first

Không có “AI terminal” riêng. CLI workflow hiện tại của bạn giữ nguyên — 8sync chỉ wrap quanh.

Context-aware

Đi tới project nào, AI bám theo ngữ cảnh đó. Memory shared qua AGENTS.md + agents/*.

Skill injection

8sync skill add <github-url> clone vào cả global ~/.omp/skills/ và project agents/skills/.

Small & native

1 binary Rust ≈ 3.8 MB stripped (gồm 15 bundled skill). Phù hợp CachyOS/Arch + Kitty + Helix + omp.

Quick start

CachyOS / Arch Linux, terminal Kitty (bật allow_remote_control yes để có layout 3-pane).

bash 
git clone https://github.com/8-Sync-Dev/su-code.git
cd su-code
bash scripts/bootstrap.sh        # rustup + cargo build + install vào ~/.local/bin

8sync setup --dry-run            # xem plan trước
8sync setup                      # cài harness + chọn profile y/N
8sync doctor                     # verify

cd ~/code/my-app
8sync .                          # mở session (kitty 3-pane + omp trong abduco)

Đảm bảo ~/.local/bin trong $PATH. Cờ thường dùng: --dry-run, --no-profile, --yall, --profile <name>.

Lệnh chính

Mọi verb có -h / --help với block EXAMPLES. Chạy 8sync hoặc 8sync help để xem cheatsheet.

Skill injection

Skill = module hướng dẫn cho AI. 8sync bundle sẵn 15 skill (codegraph · karpathy · ponytail · assp · impeccable · taste · 8sync-cli · image-routing · …); 8sync skill add collection-aware kéo thêm cả bộ từ GitHub.

bash 
# Clone từ URL: cài vào CẢ ~/.omp/skills/ (omp đọc global) lẫn agents/skills/ (project)
# Đồng thời rewrite block <!-- 8sync:skills:* --> trong AGENTS.md
8sync skill add https://github.com/colbymchenry/codegraph

# Các dạng spec khác
8sync skill add gh:owner/repo
8sync skill add path:/abs/local/dir
8sync skill add builtin:karpathy

# Idempotent: chạy lại = git pull --ff-only
8sync skill add https://github.com/colbymchenry/codegraph   # → "updated"

# Dựng / refresh toàn bộ harness (thay `skill sync` cũ)
8sync harness init
8sync harness up --timer 30m

Update

Hai cách:

Auto (khuyên dùng)

8sync up

Self-update binary (GitHub release) + omp update. Không đụng pacman -Syu.

Manual (build từ source)

cd su-code && git pull
cargo build --release
install -m755 target/release/8sync \
  ~/.local/bin/8sync
